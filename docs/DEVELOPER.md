# Developers Guide

1. Clone the repo

   ```shell
   git clone https://github.com/akikanellis/github-action-continuous-release-test
   ```
   ```shell
   cd github-action-continuous-release-test
   ```

2. Install the necessary software

* Software 1
* Software 2

3. List all commands

   ```shell
   make help
   ```

4. Lint the project

   ```shell
   make lint
   ```

5. Run the tests

   ```shell
   make test
   ```

6. Run

   ```shell
   make run
   ```
