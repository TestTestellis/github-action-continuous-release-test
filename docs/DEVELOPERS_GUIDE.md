## Useful Commands

1. List all commands

   ```shell
   make help
   ```

2. Install project dependencies

   ```shell
   make install
   ```

3. Lint the project

   ```shell
   make lint
   ```

4. Run the tests

   ```shell
   make test
   ```

5. Run the app

   ```shell
   make run
   ```

6. Clean generated files

   ```shell
   make clean
   ```
